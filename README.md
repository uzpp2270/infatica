## How the Dashboard Works

The user interface is browser-based. You log in, generate proxy lists (each list = one IP with 1,000 ports representing different proxies), configure your geo-targeting, set rotation intervals, and grab your credentials. There's also a GeoDNS-based API endpoint (`pool.infatica.io`) for API-style proxy access — you append targeting parameters to the username string, standard stuff for any developer who's worked with proxy providers before.

The dashboard handles things like checking remaining traffic, triggering manual IP rotation, and switching between proxy lists. One G2 reviewer noted it covers what you need but has a learning curve if you're not already familiar with proxy management tools — Infatica's own support team helped them get set up.
